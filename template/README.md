# Template

## Getting Started

Unzip `ScriptedConnectorTemplate.zip` and use the resulting folder as the starting point for your scripted connector.

## Before You Ship

Once your scripted connector is ready, make sure to update:

- **`README.md`** — describe what your connector does
- **`manifest.json`** — fill in the correct `name`, `id`, `version`, `description`, and `author` under the `project` key
