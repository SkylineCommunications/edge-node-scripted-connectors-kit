# Hello World Example

This is a minimal hello-world example designed to verify that your edge node is correctly set up and functioning properly.

## Purpose

Use this example to test your edge node installation and configuration. It provides a simple baseline to ensure that all components are working as expected before deploying more complex connectors.

## Getting Started

1. Deploy this example to your edge node
2. Verify that the connector initializes without errors
3. Check the logs to confirm successful execution

If this example runs successfully, your edge node is ready for production use.
